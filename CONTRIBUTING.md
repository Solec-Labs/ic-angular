# Contributing

1. Clone the repository
1. Install npm modules
   ```shell
   $ npm i
   ```
1. Install pip
1. Install [Commitizen](https://commitizen-tools.github.io/commitizen/)
1. Commence hacking
1. Stage your changes
   ```shell
   $ git add .
   ```
1. Commit your changes
   ```shell
   $ cz commit
   ```
